API Documentation

Base URL

https://your-api-domain.com/api/v1

Authentication

Most endpoints require authentication. Include your API key in the request header:

Authorization: Bearer YOUR_API_KEY

Note: User registration and login endpoints don't require authentication.

Products

GET /api/products Requires Auth
Get All Products

Retrieve a list of all products with optional filtering and pagination.

Parameters:
Parameter Type Description
limit integer Number of products per page (default: 10)
page integer Page number (default: 1)
category string Filter by category ID
sort string Sort by field (price, name, created_at)
order string Sort order (asc, desc)
Example Response:
{
  "success": true,
  "count": 25,
  "pagination": {
    "next": { "page": 2, "limit": 10 },
    "prev": null
  },
  "data": [
    {
      "id": 1,
      "name": "Product 1",
      "description": "Product description",
      "price": 99.99,
      "category": { "id": 3, "name": "Electronics" },
      "stock": 50,
      "images": ["image1.jpg", "image2.jpg"],
      "createdAt": "2023-01-01T00:00:00.000Z"
    }
  ]
}
GET /api/products/:id Requires Auth
Get Single Product

Retrieve details of a specific product.

Example Response:
{
  "success": true,
  "data": {
    "id": 1,
    "name": "Product 1",
    "description": "Product description",
    "price": 99.99,
    "category": { "id": 3, "name": "Electronics" },
    "stock": 50,
    "images": ["image1.jpg", "image2.jpg"],
    "createdAt": "2023-01-01T00:00:00.000Z"
  }
}

Users

POST /api/users/register
Register User

Create a new user account.

Request Body:
{
  "name": "John Doe",
  "email": "john@example.com",
  "password": "securepassword123",
  "role": "user"
}
POST /api/users/login
Login User

Authenticate user and receive JWT token.

Request Body:
{
  "email": "john@example.com",
  "password": "securepassword123"
}
Example Response:
{
  "success": true,
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "id": 1,
    "name": "John Doe",
    "email": "john@example.com",
    "role": "user"
  }
}

Orders

POST /api/orders Requires Auth
Create Order

Place a new order.

Request Body:
{
  "items": [
    { "product": "product_id_1", "quantity": 2 },
    { "product": "product_id_2", "quantity": 1 }
  ],
  "shippingAddress": {
    "address": "123 Main St",
    "city": "New York",
    "postalCode": "10001",
    "country": "USA"
  },
  "paymentMethod": "credit_card"
}
GET /api/orders/myorders Requires Auth
Get User Orders

Retrieve all orders for the authenticated user.

Carts

All cart endpoints require authentication. Use the /api/carts route for cart operations.

Reviews

All review endpoints require authentication. Use the /api/reviews route for review operations.

Wishlists

All wishlist endpoints require authentication. Use the /api/wishlists route for wishlist operations.

Error Responses

Common Error Responses
400 Bad Request
{
  "success": false,
  "error": "Validation Error",
  "message": "Name is required"
}
401 Unauthorized
{
  "success": false,
  "error": "Unauthorized",
  "message": "Not authorized to access this route"
}
404 Not Found
{
  "success": false,
  "error": "Not Found",
  "message": "Product not found with id of 999"
}
